# Bible Notebook App

A simple, mobile-friendly Streamlit web app to organize Bible verses by topic.  
You can create topics (folders), add verses with headers, content, and personal notes, and view them in a notebook-style layout.

---

## Features

- Add new topics (folders) for organizing verses
- Add new verses/pages with:
  - Verse title/reference
  - Verse content
  - Personal notes
- View existing verses in collapsible cards
- Mobile-friendly layout, works on iPhone and other devices
- Color-coded folders and app-like styling
